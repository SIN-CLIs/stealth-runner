### 1. рЯЖХ Qualtrics вАФ Different from TolunaStart!
**Qualtrics surveys** (`eu.qualtrics.com/jfe/form/`) use completely different DOM:

| Aspect | TolunaStart | Qualtrics |
|--------|-------------|-----------|
| Page advance | `document.querySelector("button").click()` | `document.querySelector(".NextButton").click()` |

| Checkbox | `.cf-checkbox` вАФ JS `.click()` | `input[type=checkbox]` вАФ `.click()` works |
| Text input | `input[type=number]` | `textarea.InputText` + Event dispatch |
| Matrix | `.cf-radio` (global) | `table.ChoiceStructure` (per-row radio) |
| JS required | No | Yes вАФ webpack SPA, wait 3-5s |
| Inner HTML | Normal | Shows loader, use `innerText` for content |
